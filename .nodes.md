# Bulid & Deploy Agent Server

```
docker build -t agent-platform-python-server:latest .
docker build --platform linux/amd64 -t agent-platform-python-server:latest 
```

```
kind load docker-image agent-platform-python-server:latest --name agent-platform
```

# Useful Commands

```
kubectl get pods -l app=agent-worker
kubectl delete pod -l app=agent-worker
```

# Apply the CR & CRD

```
kubectl apply -f config/crd/bases/agentplatform.example.com_multiagentflows.yaml
```

```
kubectl apply -f config/samples/multiagentflow-sample.yaml
```








# Build & Deploy Controller

```
docker build --no-cache -t agent-platform-controller:latest .
```

```
kind load docker-image agent-platform-controller:latest --name agent-platform
```

```
kubectl apply -f config/manager/service_account.yaml
kubectl apply -f config/manager/manager.yaml
kubectl apply -f config/rbac/role.yaml
```

```
kubectl get pods -l app=agent-platform-controller
```
